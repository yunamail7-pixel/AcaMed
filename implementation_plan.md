# 長安生技 (AcaMed) 網站重新設計方案

## 1. 設計理念 (Design Philosophy)
*   **專業感**：強化「陽明大學研發背景」，使用學術感的配色與排版。
*   **純淨自然**：強調「天然草本」與「低溫萃取」，加入大量自然與科技融合的視覺元素。
*   **現代交互**：捨棄舊版的表格佈局，採用響應式卡片流與平滑滾動動畫 (Framer Motion)。

## 2. 核心技術棧
*   **框架**：React + Vite (極速開發體驗)
*   **樣式**：Tailwind CSS (自定義品牌色系統)
*   **圖標**：Lucide React (現代線條風格)
*   **動畫**：Framer Motion (增強視角吸引力)

## 3. 已實現功能
*   **響應式 Navbar**：滾動透明度變化，支持移動端漢堡選單。
*   **多層次 Hero Section**：強大的視覺衝擊與明確的 CTA。
*   **產品展示矩陣**：模組化的產品展示方案。
*   **品牌故事與研發優勢**：清晰展示公司的科學根基。
*   **分層配色系統**：
    *   `primary`: #1b4d2e (深森林綠 - 代表穩重與草本)
    *   `secondary`: #2196f3 (科技藍 - 代表科學與創新)
    *   `nature`: #f9fbf9 (乳白色 - 提供清新背景)

## 4. 後續建議
*   **圖片替換**：目前的圖片為 Unsplash 佔位圖，建議更換為長安生技實際的研發室與產品實拍圖。
*   **產品詳細頁**：目前的產品矩陣點擊後可跳轉至詳細說明頁（需進一步開發內容）。
*   **亞馬遜鏈接**：在產品頁面加入直接跳轉亞馬遜商城的功能。
